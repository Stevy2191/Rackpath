# Screenshots

This directory holds the screenshots referenced from the root [README](../../README.md).

Add the following images (PNG, ~1280px wide recommended) and they'll show up
automatically in the main README's Screenshots section:

| File              | Used for                                   |
|-------------------|---------------------------------------------|
| `dashboard.png`   | Project Dashboard — site overview, warnings, activity log |
| `topology.png`    | Network Topology canvas                     |
| `rack-builder.png`| Rack Builder front/back views               |
| `devices.png`     | Device Inventory list/detail view           |
| `integrations.png`| Integrations page (UniFi, Zabbix, etc.)     |

Until these are added, the README displays broken-image placeholders with
the captions above — replace them with real screenshots from your instance.
